核心能力收束在一个 JVM 应用里

CodeAuto 的设计重点是可审计、可恢复和可扩展。它不是 IDE，而是终端里的 agent runtime。

UI

CLI + TUI 双模式

普通 CLI 适合快速对话，全屏 TUI 提供 transcript、prompt、状态栏、权限审批和斜杠菜单。

Δ

文件修改可审计

写入、编辑、patch 和 modify 统一进入 diff review，降低模型直接改文件的风险。

↯

本地命令可控

危险命令需要审批；权限支持 once、turn、always、deny with feedback 和通配规则。

M

持久化记忆

记忆以 frontmatter Markdown 保存，支持 user、feedback、project、reference 四类，并注入 system prompt。

save_memory list_memory delete_memory stale marker
∞

Skills 与 MCP

支持本地 Skills 发现、stdio MCP、Streamable HTTP MCP，以及 resources/prompts helper 工具。

load_skill mcp list mcp login backed tools

启动与日常命令

Maven 直接启动即可开发调试；打包后也可以通过脚本运行。Windows bat 会自动回到项目根目录作为 workspace。

离线 TUI

不需要 API Key，适合验证界面和工具循环。

mvn exec:java "-Dexec.args=--mock --tui"

真实模型 TUI

使用配置好的模型运行，AI 文本回复会流式显示。

mvn exec:java "-Dexec.args=--tui"

打包运行

构建 shaded JAR，或使用启动脚本进入 TUI。

mvn package -DskipTests
bin/codeauto.bat --tui

TUI 快捷命令

在全屏界面中直接管理模型、权限、记忆、会话和 MCP。

/model <name>  /memory  /permissions  /mcp

会话、记忆和指令长期可用

每次新会话都会加载多级指令；相关记忆会进入 system reminder；会话按 workspace 保存并支持恢复、分叉和压缩。

01

多级指令

加载 `~/.claude/CLAUDE.md`、`~/.codeauto/CLAUDE.md`、项目 `CLAUDE.md` 和 `CLAUDE.local.md`。

02

会话存储

会话保存到 `~/.codeauto/projects/`，支持 resume、fork、rename、compact 和跨项目浏览。

03

上下文压缩

优先使用 provider usage，本地估算兜底；上下文紧张时自动压缩中间消息。

模块边界清晰，方便继续演进

核心循环、模型适配器、权限、工具、MCP、Skills、会话 和记忆系统分别维护，测试覆盖当前主要行为。

AL

AgentLoop

驱动模型响应、工具调用、工具结果回填和最终回复。

TR

ToolRegistry

统一注册内置工具与 MCP backed tools。

PM

PermissionManager

集中处理命令、文件编辑和持久化权限规则。

MM

MemoryManager

保存、检索、删除和注入持久化记忆。

IL

InstructionLoader

将用户、应用和项目级指令组织进 system prompt。

UI

TUI Renderer

基于 JLine raw mode 和 ANSI 绘制全屏终端界面。
